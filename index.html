Modular LIMS based on R Shiny - Demo videos

A customizable web-based Laboratory Information Management System developed for laboratory traceability and data management.

Annabel Margalef Burgos

This project presents a modular Laboratory Information Management System (LIMS) built entirely in R Shiny, designed to streamline laboratory operations through three integrated modules: document management, inventory control, and sample traceability. The system offers a flexible, cost-effective alternative to commercial LIMS platforms, adaptable to the specific workflows of research and clinical laboratories.

01

Sample Manager

The Sample Manager provides complete traceability of biological samples throughout the analytical workflow.

Main functionalities

  • Sample registration
  • Status monitoring
  • Traceability
  • Barcode identification
  • Report generation
02

Inventory Manager

The Inventory Manager monitors laboratory materials and reagents while facilitating stock control.

Main functionalities

  • Stock monitoring
  • Low-stock alerts
  • Reagent management
  • Order tracking
  • Search and filtering
03

Document Manager

The Document Manager centralizes laboratory documentation and ensures version control and organized storage.

Main functionalities

  • Version control
  • Folder hierarchy
  • Metadata management
  • Document search
  • Easy updates
Developed in R Shiny
